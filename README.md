# 📦 CampusPeer

**신뢰할 수 있는 교내 중고 거래 플랫폼**

CampusPeer는 대학생들이 안심하고 물건을 거래할 수 있도록 설계된 학교 인증 기반의 중고 거래 모바일 어플리케이션입니다.

---

## 🚀 주요 기능

### 1. 사용자 인증 (2단계 보안)
- **이메일 인증**: 학교 계정(@konkuk.ac.kr 등)을 통한 기본적인 본인 확인.
- **학생증 인증**: 학생증 사진 업로드 후 관리자 최종 승인 프로세스를 통해 신뢰도 확보.

### 2. 스마트 게시글 서비스
- **카테고리 필터링**: 전자 기기, 도서, 패션 등 품목별 효율적인 검색 지원.
- **위치 기반 서비스 (Naver Map API)**: 캠퍼스 지도를 활용하여 거래 희망 장소를 명확하게 설정 및 공유.

### 3. 실시간 소통 및 평점 시스템
- **실시간 채팅 (Firebase Firestore)**: 구매자와 판매자 간의 즉각적인 대화 창구 제공.
- **거래 상태 관리**: 거래 가능, 예약 중, 거래 완료 상태를 실시간으로 업데이트.
- **상대방 평점 남기기**: 거래 완료 후 판매자/구매자에 대한 평점을 기록하여 매너 거래 유도.

---

## 🛠 Tech Stack

- **Platform**: Android (Kotlin)
- **Backend**: Firebase (Authentication, Firestore, Storage)
- **Map API**: Naver Map SDK
- **Design**: Figma

## 📈 기대 효과
- **사기 최소화**: 학교 인증을 거친 사용자들만 활동하므로 거래 안정성 극대화.
- **효율적인 검색**: 학과별, 품목별 필터링을 통해 필요한 물건을 빠르게 탐색.
- **편의성**: 위치 기반 장소 설정으로 캠퍼스 내 대면 거래를 더욱 편리하게 지원.
